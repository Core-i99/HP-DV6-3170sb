# Instructions

## SSDTTime
- Download [SSDTTime](https://github.com/corpnewt/SSDTTime)
#### macOS
-   Run the SSDTTime.command
-   Give permissions for for SSDTTime to run in system preferences<br/>
#### Windows
-   Run SSDTTime.bat<br/>
